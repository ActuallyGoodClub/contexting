# contexting

Rust core library for template-based prompt assembly, with Python and JavaScript bindings.

## Structure

- `src/` — Rust core library (`contexting-core`)
- `bindings/python/` — PyO3 + maturin Python bindings
- `bindings/js/` — wasm-bindgen WASM/JS bindings

## js_depr/

**Do not read, edit, or run anything inside `js_depr/` unless explicitly asked.**

This folder contains the original TypeScript implementation kept for reference only.
